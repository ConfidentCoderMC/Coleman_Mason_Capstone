Data Types

Python identifies data according to its type. Here are a few well-known types:

Integers
Positive and negative counting numbers
Examples: -1, 0, 1, 2
Floats
Positive and negative decimal values
Examples: -5.746484838, 3.0
Strings
Characters (letters, numbers, symbols or whitespace) in between single quotes (' ') or double quotes (" ")
Examples: "Hello", 'dog'
Booleans
Values for storing true and false values
Examples: True, False
Lists
Collection of values within square brackets ([ ]), can include multiple data values (integers, strings, etc.)
Example: ['dog', 5, True, 3.0]
Dictionaries
Collection of key:pair values within curly brackets ({ }), can include multiple data values
Example: {"color": "red", "speed": 5, "health": 20}

Variables

These data types can be assigned labels called variables with the assignment operator (=).

Example: alien = {"color": "red", "speed": 5, "health": 20}


Once you assign a variable to a piece of data, you can refer to that value with its variable name.

        
            number1 = 5
            print(number1)

This Python program will print or display the value of number1, which is 5.


Rules for Writing Variable Names:

  • Names can only have letters, numbers, and underscores
  • Name must start with a letter or underscore
  • It's good practice to start names with a lowercase letter
  • Distinguish words using camelCase or snake_case (firstNumber or first_number)
  • Make names descriptive and relevant to the value
