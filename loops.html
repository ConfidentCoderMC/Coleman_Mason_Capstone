Loops

Computers are quite efficient with repetitive tasks. In Python, it is possible to loop through characters in a string or items in a list.

for Loops

If you know how many times a loop needs to be repeated, use for loops to get the job done. The easiest way to make sure a loop repeats a specific number of times is with the range() function. The following example would execute ten times:

    
      for i in range(10):
        print("Hello World!")

while Loops

while loops are effective when the programmer unsure how many times a loop will repeat. The loop will terminate or end once the condition it checks for after each repeat becomes false. Be sure to update the condition after each repeat, or you will create an infinite loop that goes on forever. Here is the syntax for a while loop that prints "Hello World" until it has done it four times.

    
      number = 0
      while number < 4:
        print("Hello World!")
        number++
