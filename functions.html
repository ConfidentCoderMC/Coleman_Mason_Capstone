Functions

Functions are blocks of code that can be reused throughout a program; this, the program is better organized. Python has several built-in functions such as print() and range(). However, there may be times when the programmer must define their own functions.

This is accomplished by using the def keyword. Here's the basic syntax:


    def function_name(parameter(s)):
      statements

A parameter is piece of information the function needs to work, and a function can have one, none or many. If there are many, each parameter must be separated by a comma.

Once a function is finished with its work, it can return a value if there is a return statement at the very end of a function definition block along with the value or variable that needs to be returned.

However, if no value needs to be returned, the return statement can be omitted.
