Selection

Sometimes, a program needs to determine if a condition is true or false before it continues. This is where selection is useful.The if statement is the simplest form of selection and can also have an optional else clause if there are two possible flows of execution.

If there are more than two possible flows of execution, use elif statements for other branches of execution.

Here's the syntax for selection with 3 possible paths:


    if condition:
      statements
    elif condition:
      statements
    else:
      statements

The conditional statements that go after the if and elif keywords are boolean expressions that evaluate to a boolean value (True or False).

It is common to compare values and modify the behavior of the program depending on the result. Comparison operators are used for this purpose. The table below describes them in detail.


Comparison Operators
Operator Description Example
== Compares values for equality 2 == 2 (True)
1 == 2 (False)
!= Compares values for inequality 2 != 5 (True)
5 != 5 (False)
> Checks if first value is greater than the other value 4 > 1 (True)
4 > 10 (False)
< Checks if first value is less than the other value 10 < 21 (True)
9 < 5 (False)
>= Checks if first value is greater than or equal to the other value 4 >= 1 (True)
4 >= 4 (True)
6 >= 7 (False)
<= Checks if first value is less than or equal to the other value 7 <= 10 (True)
7 <= 7 (True)
8 <= 6 (False)
